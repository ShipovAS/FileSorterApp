Проект собран на Delphi XE.

Генератор файлов:
 - Принимает на вход словарь - текстовый файл. Предполагается по одному слову в строке.
   Для примера использован роман А.С. Пушкина "Евгений Онегин", поэтому в генераторе лучше использовать параметр
   "Количество слов в строке" - 1 (т.е. одна строка из словаря соответствует строке в выходном файле).
   
Сортировщик файлов:
 - Сортирует выбранный файл.
   Временные файлы - в c:\Users\<username>\AppData\Local\Temp\FileSorter_<датавремя>
